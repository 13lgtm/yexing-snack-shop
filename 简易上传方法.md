# 🚀 最简单的GitHub上传方法

如果Git命令行配置有问题，这里有一个**完全不需要Git**的简单方法！

## 方法：直接在GitHub网站上上传

### 步骤 1: 打包项目文件

1. 打开文件资源管理器，导航到：`C:\Users\Zeng\Desktop\ui\前端`
2. 右键点击 `业兴小吃` 文件夹
3. 选择 **"发送到" → "压缩(zipped)文件夹"**
4. 会生成一个 `业兴小吃.zip` 文件

### 步骤 2: 在GitHub上传

1. 登录 https://github.com
2. 访问您的仓库：https://github.com/13lgtm/yexing-snack-shop
3. 点击 **"uploading an existing file"** 或 **"Add file" → "Upload files"**
4. 将压缩包拖拽到网页，或点击 "choose your files"
5. 等待上传完成
6. 在底部 "Commit changes" 填写：`Initial commit`
7. 点击绿色按钮 **"Commit changes"**

**缺点**：这个方法一次性上传，以后更新代码会比较麻烦。

## 更好的方案：使用VS Code内置Git

VS Code 有内置的 Git 支持，不依赖系统 PATH！

### 步骤 1: 在VS Code中初始化Git

1. 在VS Code中，点击左侧的 **源代码管理** 图标（或按 Ctrl+Shift+G）
2. 点击 **"初始化仓库"** 或 **"Initialize Repository"**

### 步骤 2: 配置Git用户信息

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 `Git: Open Git Config`
3. 在打开的配置文件中添加：

```
[user]
    name = Zeng
    email = 1361828457@qq.com
```

### 步骤 3: 提交所有文件

1. 在源代码管理面板，会看到所有文件
2. 点击文件列表上方的 **"+"** 号（暂存所有更改）
3. 在顶部输入框输入：`Initial commit: 业兴小吃订餐系统`
4. 点击 **"✓ 提交"** 按钮

### 步骤 4: 添加远程仓库并推送

1. 按 `Ctrl+Shift+P`，输入：`Git: Add Remote`
2. 输入远程仓库 URL：`https://github.com/13lgtm/yexing-snack-shop.git`
3. 输入名称：`origin`
4. 按 `Ctrl+Shift+P`，输入：`Git: Push`
5. 如果提示登录，使用GitHub用户名和Personal Access Token

---

## 推荐：先等Git完全安装好

**最好的办法**：
1. 完全卸载当前的Git
2. 重新下载最新版Git：https://git-scm.com/download/win
3. 安装时确保勾选 **"Add Git to PATH"**
4. 重启电脑
5. 然后再用命令行上传
